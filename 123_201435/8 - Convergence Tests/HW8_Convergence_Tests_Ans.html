For problems 1 – 9, apply the Divergence Test. What, if any, conlcusions can you draw about the series?

  1. \(\sum_{k=1}^{\infty}(-1)^k\)

    1\(\lim_{k \rightarrow \infty}(-1)^k\) DNE and thus is not 0, so by the Divergence Test the series diverges.

    Also, recall that this series is a geometric series with ratio \(r=-1\), which
    confirms that it must diverge.

  2. \(\sum_{k=1}^{\infty}(-1)^k \text{ } \frac{1}{k}\)

    \(\lim_{k \rightarrow \infty}(-1)^k \text{ } \frac{1}{k}=0\), so the Divergence Test is inconclusive.

  3. \(\sum_{k=3}^{\infty}\frac{\ln{k}}{k}\)

    \(\lim_{k \rightarrow \infty}\frac{\ln{k}}{k}=0\), so the Divergence Test is inconclusive.

  4. \(\sum_{k=1}^{\infty}\frac{\ln{6k}}{\ln{2k}}\)

    \(\lim_{k \rightarrow \infty}\frac{\ln{6k}}{\ln{2k}}=1\neq0\) [see problem #26], so by the Divergence Test the series diverges.

  5. \(\sum_{k=1}^{\infty}{ke^{-k}}\)

    1\(\lim_{k \rightarrow \infty}{ke^{-k}}=0\) [see problem #6], so the Divergence Test is inconclusive.

  6. \(\sum_{k=1}^{\infty}\frac{e^k-e^{-k}}{e^k+e^{-k}}\)

    1\(\lim_{k \rightarrow \infty}\frac{e^k-e^{-k}}{e^k+e^{-k}}=1\neq0\) [see problem #21],

    so by the Divergence Test the series diverges.

  7. \(\sum_{k=1}^{\infty}\left(1+\frac{1}{k}\right)^k\)

    1\(\lim_{k \rightarrow \infty}\left(1+\frac{1}{k}\right)^k=e\neq0\) [see problem #34],

    so by the Divergence Test the series diverges.

  8. \(\sum_{k=1}^{\infty}(\sqrt{k^2+8k-5}-k)\)

    1\(\lim_{k \rightarrow \infty}(\sqrt{k^2+8k-5}-k)=4\neq0\) [see problem #28],

    so by the Divergence Test the series diverges.

  9. \(\sum_{k=2}^{\infty}(\sqrt{k^2+3}-\sqrt{k^2-4})\)

    \(\lim_{k \rightarrow \infty}(\sqrt{k^2+3}-\sqrt{k^2-4})=0\), so the Divergence Test is inconclusive.; Detailed Solution:

  1. \(\sum_{k=3}^{\infty}\frac{\ln{k}}{k}\)

  2. \(\sum_{k=1}^{\infty}{ke^{-k}}\)

    The series converges by the Integral Test.

  3. \(\sum_{k=1}^{\infty}{\left(\arctan\left(\frac{1}{k}\right)-\arctan(k)\right)}\)

    The series diverges by the Divergence Test. [see problem #33.]

  4. \(\sum_{k=1}^{\infty}{\frac{1}{\sqrt[4]{k+15}}}\)

    \(\sum_{k=1}^{\infty}{\frac{1}{\sqrt[4]{k+15}}}=\sum_{k=16}^{\infty}{\frac{1}{\sqrt[4]{k}}}\), which is a \(p\)-series with \(p=\frac{1}{4}<1\), so the series diverges.

  5. \(\sum_{k=1}^{\infty}{\pi^k e^{-k}}\)

    1The series diverges by the Divergence Test. Also, observe that this is a geometric series with ratio \(r=\textstyle \frac{\pi}{e}>1\), which confirms that the series diverges.

  6. \(\sum_{k=2}^{\infty}{\frac{1}{4k^2}}\)

    The series is a constant multiple of a \(p\)-series with \(p=2>1\), so the series converges.

  7. \(\sum_{k=2}^{\infty}{\frac{k^2}{4k^2+9}}\)

    The series diverges by the Divergence Test.

  8. \(\sum_{k=2}^{\infty}{\frac{k}{4k^2+9}}\)
