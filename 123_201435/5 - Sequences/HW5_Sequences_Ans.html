    1 \(\left\{ \frac{1}{4^n} \right\}_{n = 1}^{+\infty}\). Note that any integer can be used as the lower index for a sequence. So you could describe this sequence with \(\left\{ \frac{1}{4^{n+1}} \right\}_{n = 0}^{+\infty}\) or even \(\left\{ \frac{1}{4^{n+10}} \right\}_{n = -9}^{+\infty}\), although admittedly it is weird to use \(-9\) as a starting index unless there is some compelling reason to do so.       \(L=5\).

    2. How must \(a_1\) be defined to ensure that the sequence converges? Justify your answer.



      1If \(a_1 = 5\), the sequence is \(5, 5, 5, 5, ...\), which clearly converges to 5. If \(a_1 \neq 5\), say \(a_1 = K (K \neq 5)\), then the sequence oscillates bewteen \(K\) and \(10-K\), e.g. if \(a_1 = 3\) we have \(3, 7, 3, 7, 3, 7, ...\) Such a sequence diverges.
