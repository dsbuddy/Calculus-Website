EXPECTED SKILLS:

  • Given the graph of a function \(y=f(x)\), be able to determine the limit of \(f(x)\) as \(x\) approaches some finite value (as both a one-sided and two-sided limit).

  • Know how to determine when such a limit does not exist, and if appropriate, indicate whether the behavior of the function increases or decreases without bound.

  7. For each of the following, determine whether the given statement is true or false. If the statement is false, give a specific counterexample.

    1. If \(f(x)\) is not defined at \(x=c\), then \(\lim_{x\rightarrow c}f(x)\) DNE.



      1False. For example, consider \(f(x)=\left\{\begin{array}{ll} 1 & \text{if } x<0\\ 1 & \text{if } x>0\end{array}\right.\). Then, even though \(f(0)\) is undefined, we have \(\lim_{x\rightarrow 0}f(x)=1\).

    2. If \(\lim_{x\rightarrow a^-}f(x)=L\), then \(\lim_{x\rightarrow a}f(x)=L\).



      1False. For example, consider \(f(x)=\left\{\begin{array}{ll} 1 & \text{if } x<0\\ 2 & \text{if } x>0\end{array}\right.\). Then, \(\lim_{x\rightarrow 0^-}f(x)=1\); but, \(\lim_{x\rightarrow 0}f(x)\) DNE because \(\lim_{x\rightarrow 0^+}f(x)=2\neq1\).
