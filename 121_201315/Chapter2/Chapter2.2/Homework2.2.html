EXPECTED SKILLS:

  • Know how to compute the derivative of a function using the limit definition.

  • Understand the geometric interpretation of a derivative (as the slope of a tangent line), and be able to use the derivative to help find the equation of a tangent line.

  • Understand the physics interpretation of the derivative (as instantaneous velocity).

  • Understand how the graph of a function affects the derivative.

  • If given the graph of a function, be able to make a reasonable sketch of its derivative function.

PRACTICE PROBLEMS:

  1. For each of the following problems, use the definition of the derivative to calculate \(f^{\prime}(x)\).

    1. \(f(x) = 3x\)

      3

    2. \(f(x) = 2x^2-x\)

      \(4x-1\)

    3. \(f(x) = 3\sqrt{x}\)

      \(\displaystyle \frac{3}{2\sqrt{x}}\)

    4. \(\displaystyle f(x)=\frac{1}{\sqrt{x}}\)

      \(\displaystyle -\frac{1}{2x^{3/2}}\)

    5. \(\displaystyle f(x) = \frac{1}{x-1}\)

      \(\displaystyle \frac{-1}{(x-1)^2}\)

  2. Find an equation of the tangent line to the graph of the given function at the specified value of \(x\).

    1. \(f(x) = x^3\) at \(x=2\)

      \(y=12x-16\)

    2. \(f(x) = x^2-1\) at \(x=-1\)

      \(y=-2x-2\)

  3. Find an equation of the line which is tangent to the graph of \(y=f(x)\) when \(x=3\) if \(f(3)=7\) and \(f^{\prime}(3)=-1\).

    \(y=-x+10\)

  4. Suppose the tangent line to the graph of \(y=f(x)\) at the point \((1,2)\) also passes through the point \((7,5)\). Compute \(f(1)\) and \(f^{\prime}(1)\).

    \(f(1)=2\); \(\displaystyle f^{\prime}(1)=\frac{1}{2}\)

  5. Suppose \(f(x)\) is a function such that \(f^{\prime}(x)=x^2-4\).

    1. For which value(s) of \(x\) will the graph of \(f(x)\) have horizontal tangent lines?

      \(x=2\) or \(x=-2\)

    2. For which value(s) of \(x\) will the tangent line to the graph of \(f(x)\) be parallel to the line \(y=5x-37\)?

      \(x=3\) or \(x=-3\)

    3. For which value(s) of \(x\) will the tangent line to the graph of \(f(x)\) be perpendicular to the line \(y=2x+\pi\)?

      \(\displaystyle x=\sqrt{\frac{7}{2}}\) or \(\displaystyle x=-\sqrt{\frac{7}{2}}\)

  6. Consider the graph of \(y=f(x)\) shown below.

    Arrange \(f^{\prime}(a)\), \(f^{\prime}(b)\), \(f^{\prime}(c)\), \(f^{\prime}(d)\), and \(f^{\prime}(e)\) in increasing order.

    \(f^{\prime}(a)<f^{\prime}(d)<f^{\prime}(e)<f^{\prime}(b)<f^{\prime}(c)\)

  7. For each of the following, sketch the graph of the given function and determine where the function is not differentiable. Explain.

    1. \(f(x) = |x+2|\)

      1

      \(f(x)\) is not differentiable when \(x=-2\).
      The graph has a “sharp corner” at this point.

    2. \(\displaystyle f(x) = \sqrt[3]{x}\)

      1

      \(f(x)\) is not differentiable at \(x=0\).
      It has a vertical tangent line at this point.

    3. \(\displaystyle f(x)=\left\{\begin{array}{lll} x+1 & \text{if} & x>1\\ x^2 & \text{if} & x \leq 1 \end{array}\right.\)

      1

      \(f(x)\) is not differentiable at \(x=1\).
      The graph is not continuous at this point.

  8. Multiple Choice: If the function \(y=f(x)\) is not differentiable at \(x=0\), then which of the following MUST be true?

    1. \(f(0)\) is undefined.

    2. \(f(x)\) is NOT continuous when \(x=0\).

    3. There is a horizontal tangent line to the graph of \(y=f(x)\) when \(x=0\).

    4. There is a vertical tangent line to the graph of \(y=f(x)\) when \(x=0\).

    5. \(\lim_{h\rightarrow 0}\frac{f(0+h)-f(0)}{h}\) does not exist.

  9. Match each of the graphs for functions (a)-(d) with the appropriate graph of its derivative (i)-(iv).

    (a) (i)
    (b) (ii)
    (c) (iii)
    (d) (iv)

    Answer:

    \(f(x)\) \(f^{\prime}(x)\)
    (a) (iii)
    (b) (iv)
    (c) (ii)
    (d) (i)

  10. Sketch a function \(y=f(x)\) with the given characteristics. (There are many possible answers.)

    1. \(f^{\prime}(x) <0\) when \(x<0\); \(f^{\prime}(x) >0\) when \(x>0\); and \(f(0) = 0\).

    2. \(f^{\prime}(x) = 0\) when \(x<0\); \(f^{\prime}(x) <0\) when \(x>0\); and \(f(-1) = 3\); \(f^{\prime}(0)\) DNE.

    3. \(f^{\prime}(x) >0\) when \(x<-1\) and when \(x>1\); \(f^{\prime}(x) < 0\) when \(-1<x<1\).

    4. \(f(x)\) has a vertical tangent line when \(x=1\); \(f^{\prime}(x)>0\) for \(x < 1\); \(f(x)\) is not differentiable when \(x=-1\).
