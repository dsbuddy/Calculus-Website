EXPECTED SKILLS:

  • Be able to compute the local linear approximation of a function at a specific value.

  • Know how to use the local linear approximation to estimate a given quantity.

PRACTICE PROBLEMS:

For problems 1-4, calculate the Local Linear Approximation, \(L(x)\), for the given function at the specified value of \(x_0\). Also, sketch \(f(x)\) and \(L(x)\) over the indicated interval

  1. \(f(x)=x^2+1\), \(x_0=1\), \([0,2]\)

  2. \(f(x)=\frac{1}{2x-1}\), \(x_0=1\), \([0,2]\)

  3. \(f(x)=e^{-x}\), \(x_0=0\), \([-1,1]\)

  4. \(f(x) =\tan(x)\), \(x_0=\frac{\pi}{4}\), \(\left[0,\frac{\pi}{2}\right]\)

For problems 5-10, use an appropriate local linear approximation to approximate the following values.

  1. \((5.05)^3\)

  2. \(\sqrt{101}\)

  3. \(\sqrt[3]{28}\)

  4. \(e^{0.9}\)

  5. \(\cos{0.1}\)

  6. \(\sin{61^{\circ}}\)

  7. Show that \((1-x)^5 \approx 1-5x\) for \(x\) near 0.

  8. Show that \(\ln{(2x)} \approx 2x-1\) for \(x\) near \(\frac{1}{2}\).

  9. Consider \(f(x)=(x+1)^{13}\).

    1. Compute the Local Linear Approximation of \(f(x)\) at \(x_0=0\).

    2. Using your approximation, estimate \((0.99)^{13}\).

    3. Using your approximation, estimate \((1.01)^{13}\)

  10. Let \(f(x)=x^2\).

    1. Calculate the Local Linear Approximation, \(L(x)\), for \(f(x)\) at \(x=a\).

    2. Does \(L(x)\) overestimate or underestimate \(f(x)\) near \(x=a\)? Explain.

  11. Multiple Choice: Which of the following is the best local linear approximation for \(f(x)=\tan(x)\) near \(\displaystyle x=\frac{\pi}{4}\)?

    1. \(\displaystyle 1+\left(x-\frac{\pi}{4}\right)\)

    2. \(\displaystyle 1+ \frac{1}{2}\left(x-\frac{\pi}{4}\right)\)

    3. \(\displaystyle 1+\sqrt{2}\left(x-\frac{\pi}{4}\right)\)

    4. \(\displaystyle 1+2\left(x-\frac{\pi}{4}\right)\)

    5. \(\displaystyle 2+2\left(x-\frac{\pi}{4}\right)\)
