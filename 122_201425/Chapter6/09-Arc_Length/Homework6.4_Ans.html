PRACTICE PROBLEMS:

For problems 1-3, compute the exact arc length of the curve over the given interval.

  1. \(y=4x^{\frac{3}{2}}-1\) from \(x=\frac{1}{12}\) to \(x=\frac{2}{9}\)

  2. \(y = \frac{x^2}{2} - \frac{\ln(x)}{4}\) for \(2 \leq x \leq 4\)

  3. \(y=\frac{2}{3}(x^2-1)^{3/2}\) for \(1\leq x \leq 3\)

  4. Consider the curve defined by \(y = \sqrt{4-x^2}\) for \(0 \leq x \leq 2\).

    1. Compute the arc length on the interval \([0,t]\) for \(0 \leq t <2\). (Your arc length will depend on \(t\).)

    2. Use your answer from part (a) to compute the arc length on the interval \([0,2]\). (Hint: You will need to introduce a limit.)

    3. Confirm your answer from part (b) by using geometry.

  5. Consider \(F(x)=\int_1^x \sqrt{t^2-1} \,dt\). Compute the arc length on \([1,3]\)

  6. Consider the curve defined by \(f(x)=\ln{x}\) on \(\left[1,e^3\right]\)

    1. Set up but do not evaluate an integral which represents the length of the curve by integrating with respect to \(x\).

    2. Set up but do not evaluate an integral which represents the length of the curve by integrating with respect to \(y\).

  7. Consider the curve defined by \(f(x)=\tan{x}\) on \(\left[-\frac{\pi}{3},\frac{\pi}{4}\right]\)

    1. Set up but do not evaluate an integral which represents the length of the curve by integrating with respect to \(x\).

    2. Set up but do not evaluate an integral which represents the length of the curve by integrating with respect to \(y\).

  8. Consider the curve defined by \(y=\sin{x}\) for \(0 \leq x \leq \pi\).

    1. Set up but do not evaluate an integral which represents the length of the curve.

    2. Estimate the value of your integral from part (a) by using a Midpoint Approximation with three rectangles of equal width.
