Practice Problems:

  1. Solve the following problems by drawing a triangle.

    1. Find all possible values of \(\sin{\theta}\) and \(\cos{\theta}\) given that \(\tan{\theta}=3\)

    2. Find all possible values of \(\sin{\theta}\) and \(\tan{\theta}\) given that \(\displaystyle \cos{\theta}=\frac{2}{3}\)

    3. Find all possible values of \(\tan{\theta}\) and \(\csc{\theta}\) given that \(\displaystyle \sec{\theta}=\frac{5}{2}\)

  2. Compute the following:

    1. \(\displaystyle \cos{\theta}\) if \(\displaystyle \sin{\theta}=-\frac{3}{5}\) and \(\theta\) is in Quadrant IV.

    2. \(\displaystyle \tan{\theta}\) if \(\displaystyle \sec{\theta}=-\frac{9}{4}\) and \(\theta\) is in Quadrant III.

  3. Use the given information to find the exact values of the remaining five trigonometric functions of \(\theta\).

    1. \(\displaystyle \cos{\theta}=\frac{3}{5}\) and \(\displaystyle 0 < \theta < \frac{\pi}{2}\)

    2. \(\displaystyle \cos{\theta}=\frac{3}{5}\) and \(\displaystyle -\frac{\pi}{2} < \theta < 0\)

    3. \(\displaystyle \tan{\theta}=-\frac{1}{3}\) and \(\displaystyle \frac{\pi}{2} < \theta < \pi\)

    4. \(\displaystyle \tan{\theta}=-\frac{1}{3}\) and \(\displaystyle -\frac{\pi}{2} < \theta < 0\)

    5. \(\displaystyle \csc{\theta}=\sqrt{2}\) and \(\displaystyle 0 < \theta < \frac{\pi}{2}\)

    6. \(\displaystyle \csc{\theta}=\sqrt{2}\) and \(\displaystyle \frac{\pi}{2} < \theta < \pi\)
