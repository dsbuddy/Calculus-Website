Chapter 3.8: Graphs of the Trigonometric Functions

Expected Skills:

  • Be able to graph the trigonometric functions.

Practice Problems:

  1. Sketch the given function on the interval \([-2\pi,2\pi]\). Label all asymptotes, intercepts with the coordinate axes, and local extrema.

    1. \(\displaystyle y=\sin{x}\)



      \(x\) intercepts: \(\displaystyle \left(0,0 \right)\), \(\displaystyle \left(\pm \pi,0 \right)\), and \(\displaystyle \left(\pm 2\pi,0 \right)\)
      \(y\) intercept: \((0,0)\)

    2. \(\displaystyle y=\cos{x}\)



      \(x\) intercepts: \(\displaystyle \left(\pm\frac{\pi}{2},0 \right)\) and \(\displaystyle \left(\pm \frac{3\pi}{2},0 \right)\)
      \(y\) intercept: \((0,1)\)

    3. \(\displaystyle y=\tan{x}\)



      \(x\) intercepts: \(\displaystyle \left(0,0 \right)\), \(\displaystyle \left(\pm \pi,0 \right)\) and \(\displaystyle \left(\pm 2\pi,0 \right)\)
      \(y\) intercept: \((0,0)\)

    4. \(\displaystyle y=\cot{x}\)



      \(x\) intercepts: \(\displaystyle \left(\pm \frac{\pi}{2},0 \right)\) and \(\displaystyle \left(\pm \frac{3\pi}{2},0 \right)\)
      No \(y\) intercept

    5. \(\displaystyle y=\sec{x}\)



      No \(x\) intercepts, \(y\) intercept: \((0,1)\)

    6. \(\displaystyle y=\csc{x}\)



      No \(x\) intercepts and no \(y\) intercept

  2. Sketch each of the following functions on the interval \([0,2\pi]\). Label all asymptotes, intercepts with the coordinate axes, and local extrema. Also, determine the period.

    1. \(y=\cos{(3x)}\)



      0.6The period is \(\frac{2\pi}{3}\)

    2. \(y=3\sin{\left(\frac{x}{2}\right)}\)



      0.6The period is \(4\pi\)

    3. \(y=\frac{1}{2}\tan{\left(x\right)}\)



      0.7The period is \(\pi\)

    4. \(y=\tan{\left(\frac{x}{2}\right)}\)



      0.7The period is \(2\pi\)

  3. Sketch the region in the \(xy\)-plane which is enclosed by \(y=\sin{x}\) and \(y=2-\sin{x}\) for \(\frac{\pi}{2}\leq x \leq \frac{5\pi}{2}\).



  4. Sketch the region in the \(xy\)-plane which is enclosed by \(y=\sec{x}\) and \(y=\frac{1}{2}\) for \(-\frac{\pi}{4}\leq x \leq \frac{\pi}{4}\).



  5. Sketch \(f(x) = \sin^2(x)\) on the interval \([0,2\pi]\). Label all asymptotes, intercepts with the coordinate axes, and local extrema.
    Hint: \(f^{\prime}(x) = 2\sin{x}\cos{x}\) and \(f^{\prime\prime}(x)=4\cos^2{x}-2\)
